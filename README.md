# ICT3215-Timestomp-Detection-Framework

# Initial Virtual Environment Setup
1. Create a virtual environment first.
    ```
    py -m venv .venv
    ```

2. Next, activate the .venv.
    ```
    .venv\Scripts\activate
    ```
    | Note: VSCode's command prompt (CMD) terminal may be required instead of the default PowerShell terminal.

3. Install the required libraries in requirements.txt.
    ```
    pip install -r requirements.txt
    ```
